关于Hexo-themes-Bloomy

2023-02-27

介绍：一个简洁的hexo的主题
github仓库

Bloomy

  • 介绍：hexo主题Bloomy，页面简洁美观
  • 创作原因：想学着自己实现一个hexo主题
  • 演示地址：https://7up-hue.github.io/

使用

clone仓库

cd theme
git clone git@github.com:7UP-hue/MyBlog.git

更改默认配置

关闭highlight，到博客根目录下的_config.yml,蒋highlight下的enable设为false

# _config.yml
highlight:
  enable: false
  # line_number: true
  # auto_detect: false
  # tab_replace: ''
  # wrap: true
  # hljs: false
prismjs:
  enable: true
  preprocess: true
  line_number: true
  tab_replace: ''

修改完后清除缓存
hexo clean
hexo g

修改配置信息

可修改的信息包括：

  • menu菜单
  • 网页title
  • 个人头像，姓名，github地址，csdn地址，推特地址等
    修改方式：
  • 在主题目录下的_config.yaml文件中找到对应配置修改即可
